Terminal Commands

This project was generated with Angular CLI version 1.0.0 and angular 4.x.

  1. Open Terminal
  2. Go to your file project
  3. Make sure you have installed Angular CLI already. If not, please install.
  4. make sure docker image from Backend is built
  5. Run in terminal: ng build --prod
  6. Run in terminal: docker-compose up
  7. Navigate to http://localhost:4200/.
